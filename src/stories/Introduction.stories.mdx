# Bem-vindo(a) ao Storybook da `@tst-labs/basilar`

A `@tst-labs/basilar` é uma proposta de biblioteca de componentes contendo as bases necessárias para a construção de UIs em [**React**](https://reactjs.org/) e [**Material-UI**](https://mui.com/), que
leva em consideração os elementos (design tokens) estabelecidos pelo [Manual de Identidade Visual JT/TST](http://www.tst.jus.br/identidadevisualjt).
